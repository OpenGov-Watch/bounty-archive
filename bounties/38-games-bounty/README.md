# Bounty #38 – Games Bounty (DOT Play)

The Games Bounty incentivizes and rewards indie developers to build games on Polkadot. Game development is a specific niche that provides strong adoption of the underlying blockchain without requiring deep VC expertise.

**Funding:** **99,999 DOT remaining** out of **100,000 DOT total**

## Application Process

**Step 1 - Submission:**
Fork the repository and create an application using the provided template in the `/applications` folder. Name files as `project_name.md` and submit via pull request.

**Step 2 - Review:**
Curators issue comments and request changes on pull requests. Respond to feedback within **two weeks** or applications close due to inactivity.

**Step 3 - Delivery:**
Submit milestone deliverables to the Games Bounty Delivery repository.

## Requirements

- **Transparency**: Disclose all other funding sources (approved, denied, or ongoing)
- **Multi-sig wallet**: Use a multi-signature address with minimum three signatories for payouts
- **Team disclosure**: Clearly identify all team members and studio affiliations
- **Ecosystem contribution**: Projects should meaningfully contribute to the gaming ecosystem

## Funding Model

The Games Bounty offers **milestone-based grants paid in DOT**. Funding is primarily **retroactive**, based on agreed milestones requiring detailed planning and reporting.
