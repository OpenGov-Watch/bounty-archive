# Bounty #19 – ink!ubator (Wasm Smart Contracts)

The ink!ubator accelerates Wasm smart contract development on Polkadot, providing grants up to **$50,000-$60,000** for projects building with ink!. The program operates in cohorts, with Version 2.0 currently active though temporarily closed to new applications.

## Application Process

Fork the Ecosystem-Grants repository, create application from template, submit pull request, receive curator review. Approval requires **4 curator votes**.
