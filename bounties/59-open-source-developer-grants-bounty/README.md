# Bounty #59 – Open Source Developer Grants

Grants up to **$30,000** for open-source projects, targeting ~15 projects with **96,000 DOT** total funding.

## Primary Documentation

- GitHub organization: https://github.com/PolkadotOpenSourceGrants
- Application repo: https://github.com/PolkadotOpenSourceGrants/apply
- Delivery repo: https://github.com/PolkadotOpenSourceGrants/delivery
- Full proposal: https://docs.google.com/document/d/15Q5qvdZOBbl1_U2JxEfVyUGeTfEVxj2hRY7_m85_l2w/edit

## Curator Team (5 curators)

Diogo Mendonca (W3F), Otar Shakarishvili (JUST Open Source), Sebastian Müller (W3F), Niklas Jurij Plessing, Abhiraj Mengade. Compensation: **5% of bounty amount** (excluding W3F employees).

## Application Process

Fork repo, copy template, submit PR, respond to feedback, complete **KYC/KYB verification**.

## Requirements

Code must be open-sourced (OSI-approved licenses), no successful token sales, includes documentation.
