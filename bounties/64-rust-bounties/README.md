# Bounty #64 – Rust Bounties (OG Rust)

A newer bounty (approved early 2025) with **250,000 DOT** to incentivize Rust engineers completing tasks requested by OpenGov or the Polkadot Fellowship, specifically targeting code changes to Polkadot-SDK or system chain runtimes.

## Primary Documentation

- Subsquare referendum: https://polkadot.subsquare.io/referenda/1377
- OpenSquare voting: https://voting.opensquare.io/space/permanence/proposal/QmZxwyEn9Ad4X8u6JVKv9hNK7kVhF8YptRwVf3TcxK3kbq

## Curator Team (5 members)

Valery Gantchev/cl0w (HydraDX, PAL founder), Jakub Panik (Galactic Council CTO), Bastian Köcher (Parity, Fellowship member), Donal Murray (Parity, System Parachains lead), plus one open position. Non-Fellowship curators receive **$3,000/month**.

## Application Process

Tasks originate from OpenGov Wish for Change requests or Technical Fellowship. Curators publish Calls for Proposals, developers submit technical proposals, curators select based on quality, track record, and price competitiveness.
