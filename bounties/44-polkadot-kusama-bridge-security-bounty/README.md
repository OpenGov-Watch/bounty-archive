# Bounty #44 – Polkadot–Kusama Bridge Security Bounty

Bug bounty program with **$500,000 total** ($250k DOT + $250k KSM) offering rewards up to **$100,000 per vulnerability**. Scope expanded in 2025 to include Snowbridge and Hyperbridge.

## Primary Documentation

- Official bug bounty page: https://polkadot.com/bridges-bug-bounty/
- Full proposal: https://docs.google.com/document/d/1j-Nb7FYgSJpChgbvkbkHCM_dsjENhrh4Vqc4rNsmNGY/edit
- 2024 update: https://forum.polkadot.network/t/bridges-security-bug-bounty-2024-update/11415
- HackenProof program: https://hackenproof.com/programs/polkadot-bridge

## GitHub Repositories

- Polkadot SDK Bridges: https://github.com/paritytech/polkadot-sdk/tree/master/bridges
- Parity Bridges Common: https://github.com/paritytech/parity-bridges-common
- Snowbridge: https://github.com/Snowfork/snowbridge

## Bug Submission

- Email: bridgesbugbounty@polkadot.network
- Platform: https://hackenproof.com/programs/polkadot-bridge

## Curator Structure

5 curators (reduced from 7) including Parity, Hyperbridge, Snowbridge, Hydration representatives.

## Rewards

Based on severity up to $100k, payment after vulnerability patch, KYC required. Top 5 contributors receive Polkadot Blockchain Academy priority consideration.
