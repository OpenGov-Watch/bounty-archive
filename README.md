# Polkadot Bounty Archive

Archive of Polkadot bounty documentation.

## Structure

Each bounty has its own folder in `/bounties` following the pattern `id-name`:

```
bounties/
  19-inkubator/
  43-meetups-bounty/
  52-ux-bounty/
  ...
```
